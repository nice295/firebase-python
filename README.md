This project is to test python library for Firebase.
(https://github.com/thisbejim/Pyrebase)

Install library
---
$ python3 get-pip.py
(get-pip.py exists in PIP officail site.)

$ pip3 install pyrebase

Get test
---
$ python3 get.py 

Set test
---
$ python3 set.py 

Stream test (Listening event)
---
$ python3 stream.py
